# Prodesk IT — Sprint 1 Landing Page

A responsive landing page built for Prodesk IT's digital marketing wing.

## Live URL
https://prodesk-sprints.vercel.app

## Features
- Responsive Navbar with hamburger menu (mobile)
- Hero section with CTA button
- Services section with 3 cards (CSS Grid)
- Dark / Light mode toggle
- Sticky glassmorphism navbar
- Hover micro-interactions on cards and buttons
- Footer with social links

## Tech Stack
- HTML5
- Vanilla CSS (Flexbox + Grid)
- Vanilla JavaScript
